# Discount Calculator

A simple Python program that calculates the final price of an item after applying a discount.

## Features

- Accepts the original price of an item
- Accepts the discount percentage
- Calculates the discount amount
- Displays the final price after the discount

## Technologies

- Python 3

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/Gaius-codes/discount-calculator.git
```

2. Navigate to the project folder:

```bash
cd discount-calculator
```

3. Run the program:

```bash
python discount_calculator.py
```
